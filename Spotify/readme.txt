-- Readme --
To run the app, unzip the project first.

Install npm from https://www.npmjs.com/get-npm

Run the following command to install global packages:
npm install -g @angular/cli

cd into spotify/client and run the following command:
npm install

cd into spotify/webserver and run the following command:
npm install

Next, cd into spotify/webserver and run the following command:
npm start

Then, cd into spotify/client and run the following command:
ng serve

Finally, open localhost:4200 in the browser to view.
